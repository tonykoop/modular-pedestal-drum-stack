# Learn to Play — Modular Pedestal Drum Stack

The five drums are tuned A2, C3, E3, G3, A3 — an A-minor pentatonic with the D
omitted. Plenty of pitched percussion material maps onto this layout.

## Sub-folders

- `fingerings/` — strike-zone diagrams (no fingerings in the traditional sense,
  but mallet-target charts go here)
- `scales/` — pentatonic patterns and warm-up exercises
- `tunes/` — short pieces written for this instrument

A future run of the `sheet-music` skill can extend this with full notation,
audio renderings, and printable songsheets. See
`/mnt/c/Users/Tony/Documents/GitHub/handpan/learn-to-play/` for the reference
structure.
